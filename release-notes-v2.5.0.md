# Release Notes v2.5.0

## 🌐 多语言支持 / Multi-language Support

### 中文说明

**v2.5.0 (2026-07-09)**

本版本主要添加了多语言支持功能：

- ✅ **默认中文界面** - 默认使用中文界面，更符合国内用户习惯
- ✅ **可选英文界面** - 支持切换到英文界面，满足国际用户需求
- ✅ **语言设置保存** - 自动保存用户的语言偏好设置
- ✅ **启动加载语言** - 程序启动时自动加载上次的语言设置

### 技术细节
- 框架：.NET 8.0 Windows Forms
- 数据存储：JSON 格式（language.json）
- 兼容性：Windows 10 / 11

---

## English Description

**v2.5.0 (2026-07-09)**

This release adds multi-language support:

- ✅ **Default Chinese Interface** - Default to Chinese interface for domestic users
- ✅ **Optional English Interface** - Support switching to English for international users
- ✅ **Language Preference Saved** - Automatically saves user's language preference
- ✅ **Load Language on Startup** - Automatically loads the last used language setting

### Technical Details
- Framework: .NET 8.0 Windows Forms
- Data Storage: JSON format (language.json)
- Compatibility: Windows 10 / 11
