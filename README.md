# Automatic Printer Switcher Plus

Forked from [faisalthaheem/automatic-printer-switcher](https://github.com/faisalthaheem/automatic-printer-switcher).

根据当前前台窗口自动切换默认打印机，无需每次手动选择。  
Automatically switches the default printer based on the active foreground window.

## 更新内容 / What's new

- .NET 3.5 → **.NET 10**，兼容 Windows 10 / 11
- 二进制序列化 → **JSON** 存储
- **中文界面**，托盘菜单一键切换
- **开机自启**勾选框，主界面直接设置
- **UI 现代化**，深色标题栏
- 清理死代码和失效链接
